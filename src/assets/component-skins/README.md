# Monday Survival Component Skins

These assets are component-level UI skin layers for the Monday Survival H5 game.

They are not full-page screenshots. React should render all dynamic text, numbers, progress bars, and click targets above these layers.

## Scene Layers

- `desk-bg@2x.webp` — 852 x 1844, wooden desk background.
- `desk-props-back@2x.png` — 852 x 1844, transparent back props.
- `desk-props-front@2x.png` — 852 x 1844, transparent front props.

## Header Layers

- `header-paper@2x.png` — 780 x 220, paper ticket base.
- `header-left-tab@2x.png` — 174 x 194, green left tab and icon area.
- `header-tape@2x.png` — 310 x 92, transparent tape layer.
