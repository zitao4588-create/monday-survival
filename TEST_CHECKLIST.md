# 测试清单：今天你能熬过周一吗

## 自动检查

- [ ] 在仓库根目录运行 `pnpm build`。
- [ ] 在仓库根目录运行 `pnpm test`。
- [ ] 如涉及首屏或关键交互，运行 `pnpm e2e`。

## 本地试玩

- [ ] 运行 `pnpm dev`。
- [ ] 打开 `http://localhost:5180/`。
- [ ] 页面标题显示“今天你能熬过周一吗”。
- [ ] 第一回合显示“07:42 闹钟第三次响起”。
- [ ] 能看到能量、心情、得分。
- [ ] 点击任意选项后进入下一回合。
- [ ] 连续选择后能进入结算页。
- [ ] 结算页能点击“再活一次周一”重开。

## 移动端检查

- [ ] 375px 宽度下没有横向滚动。
- [ ] 390px 宽度下按钮文案不溢出。
- [ ] 430px 宽度下首屏内容排版正常。
- [ ] 选项按钮高度适合手指点击。
- [ ] 不依赖 hover 才能理解或操作。

## 内容检查

- [ ] 每个回合有 3 个选项。
- [ ] 每个选项都有明确的结果倾向。
- [ ] 文案没有错别字。
- [ ] 结局文案符合当前分数和状态。
- [ ] 新增剧情时优先改 `src/data/turns.ts`，不要直接写进组件。

## 发布前检查

- [ ] 构建产物来自 `dist/`。
- [ ] README 中的运行方式仍然有效。
- [ ] 当前版本没有引入后端、数据库、登录或支付。
